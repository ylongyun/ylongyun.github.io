#markdown基础语法及使用

##标题
markdown支持两种形式的标题。
在标题前面加#，一级标题加1个#，二级标题加2个#，以此类推，最多加6个#，也就是6级标题。
例如：
# 一级标题
##二级标题
###三级标题
####四级标题

显示结果

#一级标题

##二级标题

###三级标题

####四级标题

##块引用
mrarkdown使用>字符作为块引用，且可以嵌套引用
例如：
>一级引用内容
>>二级引用内容

显示效果

一级引用内容

二级引用内容

##列表
markdown支持有序列表和无序列表。

  • 无序列表使用\ 、+、-。
    例如：
    `    title1-title+title`

显示效果

  • title1
  • title
  • title
  • 有序列表使用数字加句号。
    例如：
    1.title
    2.title
    3.title

显示效果

1.title
2.title
3.title

##分割线
一行中只有三个或三个以上的 * 或 - 则会在这一行的位置上生成一条水平线。
例如：
***
-————–

显示效果

##代码块

  • 单行代码有两种表示方法

1.在代码前面缩进四个空格，即：[空格][空格][空格][空格]This is one line of code
2.`This is one line of code`

显示结果 

This is one line of code

This is one line of code

  • 多行代码也有两种表示方法

1.在每一行代码前面都缩进四个空格

  1. ```The first line of code
    The second line of code
    The third line of code
    ```

显示结果 

The first line of code  
The second line of code  
The third line of code

##链接
markdown有两种链接形式。
1.[假装这有文字](www.假装这是一个网址.com)
2.[假装这有文字][1]

[1]:www.假装这是一个网址.com
