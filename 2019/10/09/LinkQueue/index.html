初识栈

栈（stack）又名堆栈，它是一种运算受限的线性表。限定仅在表尾进行插入和删除操作的线性表。这一端被称为栈顶，相对地，把另一端称为栈底。向一个栈插入新元素又称作进栈、入栈或压栈，它是把新元素放到栈顶元素的上面，使之成为新的栈顶元素；从一个栈删除元素又称作出栈或退栈，它是把栈顶元素删除掉，使其相邻的元素成为新的栈顶元素。

刚开始学习栈时，我是充满了对新知识的好奇的。结果等我看完书上的描述，脑海之中就想到了链表，心想这东西和链表储存数据的形式是一样的呀，就是限定了对它的操作，只允许在表尾进行插入和删除节点，具体的操作和链表一摸一样，或者说比链表简单一点。白瞎了我的好奇，换汤不换药而已。因为之前写过对链表的操作了，所以今天就只写了对顺序栈的操作。如下：

1.定义顺序栈

typedef struct Stack
{
    int data[100];
    int top;
}LinkStack;

2.初始化栈

LinkStack *initStack(LinkStack *stack)//初始化栈 
{
    stack->top = -1;
    return stack;
}

3.入栈

void enterStack(LinkStack *stack)//入栈操作 
{
    if(stack->top+1>99)
        printf("栈已满");
    else
    {
        stack->top++;//栈顶后移 
        scanf("%d",&stack->data[stack->top]);//输入新成员元素 
    } 
}

4.出栈

int outStack(LinkStack *stack)//出栈操作 
{
    if(stack->top == -1)
    {
        printf("栈为空");
        return -1;
    }
    else
    {
        stack->top--;//栈顶前移
        return stack->data[stack->top+1];//返回新成员元素 
     } 
}

That is all.
